### 💡 專案簡介
這款遊戲帶領玩家深入探索孔廟文化，透過沉浸式互動體驗「八佾舞」的莊嚴韻律，並感受古人智慧的象徵——「智慧毛」的神奇效果。玩家將透過動作捕捉技術親身參與八佾舞的舞步，同時與虛實交錯的場景互動，重現古代孔廟儀式的莊重氛圍。遊戲不僅是一次文化巡禮，更是一場寓教於樂的沉浸式體驗，讓玩家在遊戲中感受傳統禮樂文化的深遠意涵。

> ⚠️ **注意**：本專案因不可抗力因素導致部分原始專案檔案損壞，目前主要作為設計概念、硬體架構與核心技術的成果展示。

### 🛠️ 技術棧
* **遊戲引擎：** Unity
* **體感體感追蹤：** Microsoft Xbox Kinect (用於擷取玩家骨架資訊並判定八佾舞姿)
* **硬體微控制器：** Arduino (用於偵測實體裝置狀態並與 Unity 進行序列通訊互動)

### 💡 About The Project
This interactive game invites players to explore Confucius Temple culture through an immersive experience of the "Ba-Yi Dance" (Eight-Row Dance) and the mystical "Wisdom Hair," a traditional symbol of ancient intellect. Utilizing motion-tracking technology, players physically participate in the ceremonial dance steps while interacting with a blended virtual-real environment that recreates the solemn atmosphere of ancient rituals. It is both a cultural journey and an engaging, educational experience that deepens players' appreciation for traditional music and ritual heritage.

> ⚠️ **Note**: Due to unforeseen circumstances, the original project files have been corrupted. This repository now serves primarily as a showcase of the design concept, hardware architecture, and core technical implementations.

### 🛠️ Tech Stack
* **Game Engine:** Unity
* **Motion Tracking:** Microsoft Xbox Kinect (For tracking skeleton data and recognizing ritual dance steps)
* **Microcontroller:** Arduino (For sensing physical object status and interacting with Unity via serial communication)
